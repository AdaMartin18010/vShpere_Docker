### 2.1 CPU配置优化

#### CPU分配

```bash
    # 配置CPU数量
Set-VM -VM "Web-Server-01" -NumCpu 4

    # 配置CPU限制
Set-VM -VM "Web-Server-01" -CpuLimitMhz 3200

    # 配置CPU预留
Set-VM -VM "Web-Server-01" -CpuReservationMhz 1600
```

#### CPU高级配置

```bash
    # 配置CPU亲和性
Set-VM -VM "Web-Server-01" -CpuAffinity "0,1,2,3"
```
