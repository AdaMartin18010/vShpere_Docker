## 模块概述

本模块详细介绍Kubernetes容器网络的完整解决方案，从CNI网络规范、主流网络插件（Calico、Cilium），到网络策略配置和网络安全。涵盖生产级容器网络所需的全部知识点。

**核心内容**:

- CNI网络规范与工作原理
- 主流CNI插件对比与选型
- Calico网络配置与BGP模式
- Cilium eBPF网络与Hubble可观测性
- NetworkPolicy网络策略与安全
