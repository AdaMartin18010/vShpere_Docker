## 模块概述

本模块详细介绍Kubernetes容器存储的完整解决方案，从CSI存储规范、主流存储方案（Rook-Ceph、Longhorn），到StorageClass配置和存储最佳实践。涵盖生产级容器存储所需的全部知识点。

**核心内容**:

- CSI存储规范与架构
- 存储类型对比与选型
- Rook-Ceph分布式存储
- Longhorn云原生存储
- StorageClass配置与管理
- 存储性能优化
