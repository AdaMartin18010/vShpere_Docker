## 模块概述

本模块提供虚拟化与容器化部署的辅助资源，包括术语表、命令速查手册、自动化脚本库、工具下载指南、参考链接汇总和常见问题解答。

**核心内容**:

- 技术术语双语对照
- 常用命令快速查询
- 自动化脚本集合
- 工具软件下载指南
- 官方文档与社区资源
- FAQ故障排查
